# 07. 카카오톡 공유 (KakaoTalk Share)

> 출처: <https://developers.kakao.com/docs/ko/kakaotalk-share/common>

사용자가 *직접* 서비스 콘텐츠(대회 카드, 추천 일정)를 카카오톡 친구·대화방에 공유. 카카오톡 메시지 API와 달리 별도 동의항목·심사 불필요.

## 카카오톡 메시지와의 차이

| 항목 | 카카오톡 공유 | 카카오톡 메시지 |
|---|---|---|
| 발송 대상 | 모든 카톡 친구·대화방 (사용자가 선택) | 친구목록 API 조건 충족 친구만 |
| UI 구현 | 불필요 — SDK가 피커 제공 | 직접 구현(피커/목록) |
| 권한 | **불필요** | 별도 사용 권한 신청 필요 |
| REST API | 미지원 (공유 링크 만들기 일부 지원) | 지원 |

## 지원 플랫폼

JavaScript SDK · Android SDK · iOS SDK · (제한적) REST API

## 메시지 템플릿 종류

- **기본 템플릿** — 사전 정의 (피드 / 리스트 / 위치 / 커머스 / 텍스트 / 캘린더)
- **사용자 정의 템플릿** — 메시지 템플릿 도구로 직접 구성
- **스크랩 메시지** — 웹페이지 URL 자동 스크랩

## 최소 카카오톡 버전

- Android 10.0.0 이상 (피커 설정은 26.2.0+)
- iOS 10.0.0 이상 (피커 설정은 26.2.0+)

## 사전 설정

1. 카카오디벨로퍼스 앱 생성 (또는 카카오 로그인 앱과 동일)
2. **카카오톡 공유 활성화**
3. 메시지 템플릿 등록 (사용자 정의 사용 시)
4. JavaScript SDK 사용 시 도메인 등록

## JavaScript SDK 사용 예시 (개념)

```html
<script src="//developers.kakao.com/sdk/js/kakao.min.js"></script>
<script>
  Kakao.init('{JS_APP_KEY}');

  Kakao.Share.sendDefault({
    objectType: 'feed',
    content: {
      title: '부산바다마라톤 2026 — 하프',
      description: '11월 둘째 주 일요일 / 광안리 출발 / 추천 D+1 코스 함께 보기',
      imageUrl: 'https://runtrip.example/marathons/busan-2026/cover.jpg',
      link: {
        mobileWebUrl: 'https://runtrip.example/marathons/busan-2026',
        webUrl: 'https://runtrip.example/marathons/busan-2026'
      }
    },
    buttons: [{
      title: '대회 보러가기',
      link: { webUrl: 'https://runtrip.example/marathons/busan-2026' }
    }]
  });
</script>
```

## 런트립 활용 매핑

| 기획안 기능 | 사용 방식 |
|---|---|
| 대회 상세 카드 공유 | `feed` 템플릿 — 일정·종목·코스·접수 마감 노출 |
| 추천 일정 (D-1/D-day/D+1) 공유 | `list` 템플릿 — 시점별 코스 |
| 동반자에게 결승 직전 합류 위치 공유 | `location` 템플릿 — 좌표 + 지도 미리보기 |
| 가족·친구 동행 모집 | `feed` + 버튼 → 가입 페이지 |

## 도입 우선순위

- **MVP**: 카카오톡 공유 — 권한 신청 불필요, 바이럴 채널로 즉시 활용
- **V2**: 카카오톡 메시지(자동 알림)로 확장
