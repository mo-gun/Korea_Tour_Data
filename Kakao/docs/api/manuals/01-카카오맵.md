# 01. 카카오맵 (Kakao Map)

> 출처: <https://developers.kakao.com/docs/ko/kakaomap/common> · <https://apis.map.kakao.com/web/guide/>

런트립의 두 번째 메인 기능 "대회 선택 → 카카오맵 기반 전·후 여행 동선 추천"의 시각화 계층.

## 개요

웹/앱 서비스에 지도, 마커, 폴리라인, 인포윈도우, 클러스터, 로드뷰 등을 임베드할 수 있도록 카카오에서 제공하는 SDK/플랫폼.

## 지원 플랫폼

| 플랫폼 | 키 종류 |
|---|---|
| **JavaScript SDK** (런트립 1순위) | JavaScript Key |
| Android SDK | Native App Key |
| iOS SDK | Native App Key |

> JavaScript SDK는 REST API Key 사용 **금지**. JavaScript Key 전용.

## 인증 / 사전 설정

1. 카카오디벨로퍼스 앱 생성
2. 플랫폼 키 정보 등록 (도메인 화이트리스트)
3. 앱 관리 페이지 → 카카오맵 사용 설정 [ON]
4. 등록 도메인에서만 SDK 동작

## SDK 로드 / 초기화

```html
<script type="text/javascript"
        src="//dapi.kakao.com/v2/maps/sdk.js?appkey={JS_APP_KEY}"></script>
```

- 상대 프로토콜(`//`) 사용 — HTTP/HTTPS 자동 대응
- SDK 스크립트는 사용 코드보다 먼저 선언되어야 함

### 라이브러리 옵션

```html
<!-- 단일 -->
<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=KEY&libraries=services"></script>
<!-- 다중 -->
<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=KEY&libraries=services,clusterer,drawing"></script>
```

| 라이브러리 | 용도 |
|---|---|
| `services` | 장소 검색, 주소-좌표 변환 (※ 별도 로컬 REST와 함께 활용 가능) |
| `clusterer` | 마커 클러스터링 — 결승선 반경 다수 POI 표시 시 |
| `drawing` | 마커·도형 직접 그리기 |

## 기본 지도 생성 예시

```html
<!DOCTYPE html>
<html>
<head><meta charset="utf-8"/><title>RunTrip Map</title></head>
<body>
  <div id="map" style="width:100%;height:400px;"></div>
  <script src="//dapi.kakao.com/v2/maps/sdk.js?appkey={JS_APP_KEY}"></script>
  <script>
    const container = document.getElementById('map');
    const options = {
      center: new kakao.maps.LatLng(35.158, 129.158), // 부산 광안리
      level: 3
    };
    const map = new kakao.maps.Map(container, options);
  </script>
</body>
</html>
```

### 주요 클래스 (요약)

- `kakao.maps.Map` — 지도 컨테이너
- `kakao.maps.LatLng(latitude, longitude)` — **위도 먼저, 경도 나중**
- `kakao.maps.Marker`, `kakao.maps.InfoWindow`, `kakao.maps.Polyline`, `kakao.maps.Polygon`
- `kakao.maps.Roadview`, `kakao.maps.MarkerClusterer` (라이브러리 필요)

## 지도 URL 스킴 (앱 호출 없이 카카오맵 웹으로 이동)

| 기능 | URL 패턴 | 예시 |
|---|---|---|
| 지도 표시 | `/link/map/위도,경도` | `https://map.kakao.com/link/map/37.395,127.110` |
| 장소명 포함 지도 | `/link/map/이름,위도,경도` | `https://map.kakao.com/link/map/카카오,37.395,127.110` |
| 길찾기 | `/link/to/이름,위도,경도` | `https://map.kakao.com/link/to/카카오,37.395,127.110` |
| 로드뷰 | `/link/roadview/위도,경도` | `https://map.kakao.com/link/roadview/37.395,127.110` |
| 검색 | `/link/search/검색어` | `https://map.kakao.com/link/search/카카오` |

> 발전 방향 단계에서 외부 공유 카드 클릭 시 카카오맵 웹/앱으로 즉시 이동하는 데 활용.

## 가격 / 쿼터

- 무료 쿼터 제공, 초과 시 유료 API 설정 필요
- 월간/일간 쿼터 적용
- 429 응답 = 쿼터 초과

## 런트립 활용 매핑

| 기획안 기능 | 카카오맵 사용 방식 |
|---|---|
| 대회 상세 카드의 "출발 좌표 + 종목별 추정 반경(원형) 시각화" | `Map` + `Circle` (또는 `Polygon`) |
| D-1 → D-day → D+1 타임라인 코스 그리기 | `Polyline` 3개 + 시점별 색상/스타일 토글 |
| 코스 영역 내 POI(맛집·찜질방 등) 마커 | `Marker` + `MarkerClusterer` (혼잡 방지) |
| POI 클릭 시 상세 정보 팝업 | `InfoWindow` + 좌측 사이드 패널 동기화 |
| 동반자 분리형 트랙 | 러너 코스/동반자 코스를 별도 레이어로 토글 (Polyline 그룹화) |
| "피니시라인 리프레시" 사우나·찜질방 시각화 | `services`/로컬 REST 결과 → `Marker` |
| 로드뷰 | `Roadview` — 결승선·랜드마크 미리보기 (UX 부가 기능) |
