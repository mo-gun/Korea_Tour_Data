# 09. 카카오톡 채널 (KakaoTalk Channel)

> 출처: <https://developers.kakao.com/docs/ko/kakaotalk-channel/common>

비즈니스 채널과 사용자의 친구 관계를 SDK/API로 관리. 발전 방향의 **B2B (지자체·소상공인 SaaS)** + **재방문 마케팅** 기반.

## 핵심 기능

- **채널 추가** — `followChannel()` (즉시 응답) / `addChannel()` (브릿지 페이지)
- **고객 관리** — 사용자 그룹화, 타겟팅 메시지
- **관계 추적** — 채널 친구 추가 상태 조회, 액션 알림

## 사전 조건

1. 카카오톡 채널 개설 + 사업자 정보 등록
2. **앱과 채널의 사업자 정보 일치** 필요
3. 사용자가 **카카오톡 채널 추가 상태 및 내역** 동의항목 동의

## 지원 플랫폼

REST API · JavaScript · Android · iOS · Flutter

## 주요 SDK 메서드

| 메서드 | 설명 |
|---|---|
| `followChannel()` | 권장. 즉시 UI + 응답 확인 가능 |
| `addChannel()` | 브릿지 페이지로 이동 — 별도로 상태 확인 호출 필요 |

## REST API (서버 전용)

문서에서 6개 핵심 엔드포인트 안내:
1. 채널 관계 확인 (단일 사용자)
2. 다수 사용자 관계 조회
3. 고객 파일 생성/관리
4. 고객 파일에 사용자 추가/제거

> 고객 관리(파일/타겟팅) 관련 엔드포인트는 **반드시 서버에서만 호출**.

## 카카오싱크 채널 추가와의 관계

카카오싱크 동의 화면 내에서 "채널 친구 추가" 옵션 노출 가능 → 회원가입 + 채널 추가가 한 번에 처리되어 마케팅 자산화 효율 최대.

## 런트립 활용 매핑

| 기획안 기능 | 사용 방식 | 단계 |
|---|---|---|
| **부산 / 강원 / 제주 지자체 채널 운영** | 지자체별 채널 → 지역 추천 일정 발송 | V2 (지역 확장 단계) |
| **회원가입 시 동시 채널 추가** | 카카오싱크 + `followChannel()` | V2 (비즈 앱 + 카카오싱크) |
| **소상공인 매출 활성화 KPI 대시보드** | 채널 친구 데이터 + KTO #16/#27~29 결합 | V3 (B2B SaaS) |
| **재방문 유도** | 채널 친구 대상 알림톡 (별도 비즈니스 메시지 상품) | V2 |

## 도입 권장 시점

- **MVP**: 사용 X (먼저 사용자 모수 확보)
- **V2**: 비즈 앱 전환 + 카카오싱크와 함께 도입
- **V3**: B2B SaaS 단계에서 채널 데이터를 KPI 산출에 결합
