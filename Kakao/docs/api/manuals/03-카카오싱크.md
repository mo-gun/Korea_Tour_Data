# 03. 카카오싱크 (Kakao Sync)

> 출처: <https://developers.kakao.com/docs/ko/kakaosync/common>

카카오 로그인 위에 얹는 **간편가입 솔루션**. 동의 화면에서 서비스 약관 + 개인정보 + (선택) 카카오톡 채널 친구 추가까지 한 번에 처리.

## 카카오 로그인과의 차이

| 구분 | 카카오 로그인 | 카카오싱크 |
|---|---|---|
| 가입 절차 | 로그인 + 자체 약관 페이지 별도 | 동의 화면에 약관까지 통합 — 단일 절차 |
| 채널 친구 추가 | 별도 호출 | 동의 화면에서 동시 처리 |
| 사용자정보 수집 폭 | 기본 항목 | 이름·이메일·전화번호·연령대·생일·성별·배송지 등 확장 |
| 사용 조건 | 일반 앱 가능 | **비즈 앱 전환 + 심사 필수** |

## 제공 기능

- **간편가입** — 동의 화면 내 약관/개인정보 수집 통합
- **다양한 사용자정보 제공** — 이름·이메일·전화번호·연령대·생일·성별·배송지 등
- **카카오톡 채널 친구 추가** — 동의 화면에서 채널 추가 요청
- **마케팅 연동** — 타겟팅 메시지/재방문 유도

## 통합 회원가입 플로우

```
사용자 [카카오로 시작하기]
  ↓
카카오톡/카카오계정 인증 + 정보 동의 (서비스 약관 + 채널 친구추가 한 화면)
  ↓
서비스가 카카오 API로 사용자정보 수령
  ↓
별도 입력 절차 없이 회원가입 완료
```

## 사전 신청·심사 절차 (필수)

1. **앱 오너/에디터 동의** — 신청 자격 확인
2. **비즈 앱 전환** — 사업자 정보 등록
3. **개인정보 동의항목 권한 신청** — 항목별 카카오 심사
4. **재검수** — 추가 사용자정보 수집 시 재심사

> MVP 일정 산정 시 비즈 앱 전환·심사 리드타임을 별도 확보할 것. 일반 카카오 로그인으로 먼저 출시 후 단계적 전환도 가능.

## 동의항목 구조

| 구분 | 정의 |
|---|---|
| **필수 동의** | 회원가입·이용에 반드시 필요한 정보. 미동의 시 가입 불가 |
| **선택 동의** | 미동의도 가입 가능. 데이터가 없을 수 있어 필수 정보로 대체 불가 |

## 런트립 활용 매핑

| 기획안 기능 | 사용 방식 |
|---|---|
| 원터치 로그인 + 회원가입 | 카카오싱크 한 화면에 통합 — 이탈률 최소화 |
| 부산·강원·제주 지자체 채널 운영 (발전 방향) | 회원가입 동시 채널 친구 추가 (선택 동의) |
| 마케팅 타겟팅 메시지 (B2B KPI) | 채널 친구 기반 알림톡·재방문 유도 |
| 동반자 분리형 트랙 / 가족 동반 | 동행자 정보 입력 단계 단축 |
