관광정보표출방법

제공되는각 API를활용하여관광정보목록및상세정보화면을기획/개발하는흐름을설명합니다.

전체적인관광정보서비스기획및개발흐름의이해를돕고자 Guide 사이트를운영하고있습니다.

추가적인정보를원하시면아래사이트를방문하시기바랍니다.

※ 개방 데이터 활용 사이트 : api.visitkorea.or.kr

인증키활용및 API호출방법

- 개발계정은일 1,000건의트래픽을제공합니다.

- 개발계정은자동승인으로활용신청후, 약 10분이후에사용이가능합니다.

- 운영계정은일 100,000건의트래픽을제공합니다.

- 운영계정은공공데이터포털 승인요청후 한국관광공사 담당자승인후사용이가능합니다.

( 운영신청시 활용 URL[웹/앱]정보를 필수등록. )

REST방식의 URL 요청예시

응답표준은XML 이며, JSON을요청할경우“&_type=json”을추가하여요청합니다.

- Json 요청 :

http://apis.data.go.kr/B551011/tursmService/empmnInfoList?serviceKey=serviceKey&numOfRows=10&pageNo=1&MobileOS=ETC&MobileApp=TestApp&_type=json

요청파라미터에서비스명기재

MobileApp파라미터는서비스(웹,앱등)별로활용통계를산출하기위한항목입니다.

URL요청시반드시기재부탁드립니다.

http://apis.data.go.kr/B551011/tursmService/empmnInfoList?serviceKey=serviceKey&numOfRows=10&pageNo=1&MobileOS=AND&MobileApp=appName

- 운영계정은 한국관광공사 담당자의 승인이 필요하며, 약 1~3일 정도 소요 됩니다.

- 승인 시 반드시 확인하는 부분은 아래와 같으며, 한국관광공사 [활용사례]에 등록되어

홍보되오니 상세히 기재하시면 됩니다.

활용 어플 URL 확인 : App/Web/기타(키오스크등)

개발계정의 TourAPI 호출 이력 및 App/Web/기타(키오스크등) 확인.

- [ 마이페이지 > 데이터활용 > Open API > 활용신청 현황 상세보기 ] 화면에서

[ 운영계정신청 ] 버튼을 통해 운영계정 신청을 할 수 있습니다.

- 운영계정 신청에서는 상세 기능정보 선택 및 필수정보를 빠짐 없이 작성 후 신청 하면 됩니다.

- 정상적인 서비스가 되고 있음이 확인 된 후 API 제공 기관인 한국관광공사 담당자가

승인을 하게 됩니다.

- 승인후 서비스 활용기간은 승인일로부터 24개월 간 활용 가능하며, 기한만료시 [연장신청] 버튼을

  통해 기한을 연장 신청을 할 수 있습니다.
