인증키 활용 및 API호출 방법

- 개발계정은 일 1,000건의 트래픽을 제공합니다.

- 개발계정은 자동승인으로 활용 신청 후, 약 10분 이후에 사용이 가능합니다.

  (공공데이터포털 연동)

REST 방식의 URL 요청 예시

응답 표준은 XML 이며, JSON을 요청할 경우 “&_type=json”을 추가하여 요청합니다.

-Json 요청:

http://apis.data.go.kr/B551011/MdclTursmService/ldongCode?serviceKey=ServiceKey&numOfRows=10&pageNo=1&MobileOS=ETC&MobileApp=TestApp&_type=json

요청 파라미터에 서비스명 기재

MobileApp 파라미터는 서비스(웹, 앱 등)별로 활용 통계를 산출하기 위한 항목입니다.

URL 요청 시 반드시 기재 부탁드립니다.

http://apis.data.go.kr/B551011/MdclTursmService/ldongCode?serviceKey=ServiceKey&numOfRows=10&pageNo=1&MobileOS=AND&MobileApp=appName
